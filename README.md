## Usage
```shell
cp .env.example .env

forge install foundry-rs/forge-std transmissions11/solmate Openzeppelin/openzeppelin-contracts --no-git --no-commit

forge build

source .env

# forge script script/xxx.s.sol:xxx --broadcast -vvvv --rpc-url $XXX_RPC_URL
```
